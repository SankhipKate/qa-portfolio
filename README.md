# QA Portfolio

This repository contains examples of my QA work: test documentation, bug reports, API testing examples, SQL practice, automation examples, and anonymized case studies.

## About Me

I am a QA Engineer with 3 years of experience in fintech and AI products.

My main experience is connected with complex business workflows, internal tools, API testing, test documentation, bug reporting, and building QA processes from scratch.

## Featured Case Study

The main case study in this portfolio is an anonymized QA case based on a retail agent onboarding system in a fintech environment.

It covers:

- working as the sole QA Engineer;
- building QA processes from scratch;
- documenting business logic and user flows;
- creating requirement coverage matrices;
- improving bug reporting quality;
- using trace IDs, evidence, and log excerpts;
- testing complex multi-step workflows with many business branches.

## Confidentiality Note

All materials in this repository are anonymized.

No real customer data, internal company documentation, production credentials, private API endpoints, financial data, or confidential business information are included.
