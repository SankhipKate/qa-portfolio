# QA Portfolio

This repository contains examples of my QA work: test documentation, bug reports, API testing examples, SQL practice, automation examples, and anonymized case studies.

## About Me

I am a QA Engineer with 3 years of experience in fintech and AI products.

My main experience is connected with complex business workflows, internal tools, API testing, test documentation, bug reporting, and building QA processes from scratch.

## Featured Case Study

- [AI-Assisted Sales Agent Hiring Flow](case-studies/ai-assisted-sales-agent-hiring-flow.md)

This case study describes an anonymized fintech mass-hiring flow for Sales Ambassadors.

It covers:

- application form validation;
- automated candidate filtering;
- city and province matching;
- vacancy availability logic;
- buffer and waitlist scenarios;
- AI recruiter interview flow;
- OIF document submission and resubmission;
- HR check statuses;
- training scheduling;
- SMS and email notification testing;
- Jira status transitions;
- UAT and production testing strategy.
